WorldVue AI

Global News Frame Analysis & Bias Detection

Project Overview

WorldVue is a political bias detection system that analyzes how different countries and media sources frame news stories. Using CORE24 (12-dimension political bias detection) and Entity Framing analysis, the platform reveals ideological positioning across economic and social/authority dimensions.

Core Innovation: CORE24 framework provides multidimensional political bias analysis across 12 dimensions organized into 4 quadrants (Economic Left/Right, Social Libertarian/Authoritarian), going far beyond simple left-right spectrum analysis.

Core Systems

CORE24 Bias Detection

12-dimension political bias analysis across economic (pro_market, pro_worker, elite_status) and social/authority (democratic_norms, inclusive, social_values) axes

Entity Framing Analysis

12-dimension analysis of how entities (politicians, organizations) are portrayed across economic, social, and meta dimensions

Interactive World Map

GeoJSON-based visualization showing country-level bias patterns with real-time filtering by topic and entity

Multilingual Search

XLM-RoBERTa powered search across articles from multiple countries and languages

RESTful API

Flask backend with endpoints for topic filtering, entity analysis, and bias detection

Real-Time Analytics

Trending topics detection and time-windowed analysis with caching for performance

Technical Implementation

Technology Stack

Python Flask XLM-RoBERTa OpenAI API Supabase PostgreSQL GeoJSON PyTorch

CORE24 Framework

The CORE24 system analyzes political bias across 12 dimensions organized into 4 ideological quadrants:

  • Economic Left: pro_market (negative), pro_worker (positive), elite_status (negative)
  • Economic Right: pro_market (positive), pro_worker (negative), elite_status (positive)
  • Social Libertarian: democratic_norms, inclusive, social_values (progressive)
  • Social Authoritarian: democratic_norms (weak), inclusive (low), social_values (traditional)

Entity Framing System

XLM-RoBERTa based multi-label classifier analyzing entity portrayal across 12 dimensions:

  • Economic Dimensions: pro_market, pro_worker, elite_status, economic_competence
  • Social/Authority Dimensions: democratic_norms, inclusive, social_values, moral_character
  • Meta Dimensions: competence, victim_aggressor, insider_outsider, threat_defender
  • Output: POSITIVE/NEGATIVE/NEUTRAL predictions with confidence scores, net score, and aggregate metrics

Architecture Overview

  • Backend: Flask server with RESTful API (topic filtering, entity analysis, multilingual search)
  • Database: Supabase PostgreSQL with entity_framing table, RLS policies, and optimized indexes
  • CORE24 Judge: LLM-based bias detection using OpenAI API with rate limiting and cost control
  • Entity Framing Predictor: Fine-tuned XLM-RoBERTa model for 12-dimension classification
  • CLI System: Command-line tools for batch judging and entity framing analysis
  • Frontend: Interactive world map with GeoJSON visualization, topic/entity filtering, and article cards

Development Journey

WorldVue represents end-to-end system design and implementation, from theoretical framework to production deployment:

Key Development Phases

  • Framework Design: Developed CORE24 12-dimension political bias framework with balanced signal bank
  • Model Training: Fine-tuned XLM-RoBERTa for 12-dimension entity framing classification (multilingual)
  • Backend Architecture: Built Flask RESTful API with Supabase PostgreSQL integration
  • Database Schema: Designed entity_framing table with RLS policies and optimized indexes
  • CLI System: Created command-line tools for batch processing and analysis workflows
  • Frontend Visualization: Implemented interactive world map with GeoJSON country-level bias visualization
  • Production Deployment: Clean production system with caching, rate limiting, and cost control

Technical Challenges & Solutions

  • Challenge: Multidimensional bias beyond simple left-right spectrum
    Solution: CORE24 framework with 4 quadrants (Economic Left/Right, Social Libertarian/Authoritarian)
  • Challenge: Entity framing across multiple ideological dimensions
    Solution: Fine-tuned XLM-RoBERTa for 12-dimension multi-label classification with confidence scores
  • Challenge: LLM API costs for large-scale bias detection
    Solution: Parallel processing with rate limiting, budget tracking, and caching strategies
  • Challenge: Real-time visualization of complex bias patterns
    Solution: GeoJSON world map with time-windowed queries and optimized database indexes

System Capabilities

API Endpoints

  • GET /api/topic/<topic>: Retrieve articles filtered by topic with bias scores
  • GET /api/country/<country>: Get country-specific articles and aggregate bias metrics
  • GET /api/trending: Discover trending topics globally with article counts
  • POST /api/entity-framing/analyze: Analyze entity framing for specific article/entity pairs
  • GET /api/entity-framing/<entity>: Retrieve all framing results for a given entity
  • GET /api/search: Multilingual article search with XLM-RoBERTa embeddings

CLI Commands

  • worldvue judge: Batch CORE24 bias detection with parallel processing and rate limiting
  • worldvue entity-framing analyze: Entity framing analysis for articles with entity extraction

Production Features: Caching layer (15-min TTL), time-windowed queries (7-day default), cost control system, RLS security policies, optimized database indexes, and parallel processing with worker pools.
